Definição

Um framework é uma estrutura abstrataque fornece suportegenérico para um software. Ele inclui bibliotecas, convenções e padrões que facilitam o desenvolvimento de aplicações.

"um framework é uma arquitetura desenvolvida com o objeto de atingir a máxima reutilização..." - Mattsson(2000)

Aplicação

Existem frameworks para várias áreas de programação, como:

  1. desenvolvimento web
  2. inteligência artificial
  3. aplicativos móveis
  4. outros

Exemplos

Alguns exemplos populares de frameworks inclui incluem:

  • React.js
  • Angular
  • Vue.js
